Research & Publications

1. Dynamic Simulation Framework for Disinformation Dissemination and Correction With Social Bots.

Link: PDF Code

Authors: Boyu Qiao, Kun Li, Wei Zhou, and Songlin Hu.

Conference: EMNLP 2025 (CCF-B, Review Stage)

Objective: Simulating disinformation dynamics to assess the impact of malicious bots and the effectiveness of correction strategies from legitimate bots.

Technical Framework:

  • Constructing a social propagation network based on the Stochastic Block (SB) and the Barabási-Albert (BA) Model.
  • Utilizing prompt engineering and statistical modeling methods to generate multi-dimensional attribute features for user agents, including identity labels, interest preferences, and behavioral patterns.
  • Establishing a time-driven mechanism and tool-calling interfaces to schedule dynamic interaction simulations among malicious bot agents, legitimate bot agents, and regular human agents at various stages.
  • Designing dynamic quantitative evaluation metrics to assess the influence of malicious and legitimate bots on regular users.

2. BotSim: LLM-Powered Malicious Social Botnet Simulation.

Link: PDF Code Dataset

Authors: Boyu Qiao, Kun Li, Wei Zhou, Shilong Li, Qianqian Lu, and Songlin Hu.

Conference: AAAI 2025 (CCF-A)

Objective: Construct a social network simulation based on LLM to model social bot strategies and develop an LLM-driven social bot dataset.

Technical Framework:

  • Build the social network simulation using the LangChain framework and Prompt engineering.
  • Use the PRAW library to collect real account data from Reddit and build the raw ecological data for the simulation environment.
  • Integrate simulation flow, bot role settings, and actions to drive Prompt-based bot agent interactions, creating a dataset with 1,907 humans and 1,000 bots.
  • Evaluate dataset quality using four baseline methods: feature engineering, semantic encoding, homogeneous GNN, and heterogeneous GNN.

3. Identifying Bots on Social Media through Coordinated Group Perception.

Link: PDF Code

Authors: Boyu Qiao, Kun Li, Wei Zhou, Shilong Li, Qianqian Lu, and Songlin Hu.

Conference: ICASSP 2025 (CCF-B)

Objective: Identify the inherent coordination behaviors of bot account groups using community detection techniques.

Technical Framework:

  • Extract coordination features from posting content using Autoencoders and K-means.
  • Uncover coordination features in social relationships with GCN and differentiable modularity.
  • Aggregate user attributes and coordination group features based on RGCN.

4. Dispelling the Fake: Social Bot Detection Based on Edge Confidence Evaluation.

Link: PDF Code

Authors: Boyu Qiao, Wei Zhou, Kun Li, Shilong Li, and Songlin Hu.

Journal: TNNLS 2024 (SCI Q1)

Objective: Identify disguised interaction edges of bots in social networks to mitigate their impact on the GNN homogeneity assumption.

Technical Framework:

  • Encode account metadata and text using MLP and RoBERTa, and fuse features with multi-head attention.
  • Construct edge features based on L1-distance between user nodes, and utilize Parametric Gaussian to reconstruct edge representations in latent semantic space.
  • Estimate edge confidence with the Sigmoid function, and remove low-confidence edges based on Bernoulli distribution to update the social graph.
  • Apply six types of homogeneous and heterogeneous GNN methods on the updated graph for node classification.

5. Multi-modal Social Bot Detection: Learning Homophilic and Heterophilic Connections Adaptively.

Link: PDF

Authors: Shilong Li, Boyu Qiao, Kun Li, Qianqian Lu, Meng Lin, and Wei Zhou.

Conference: MM 2023 (CCF A)

Objective: Enhance edge relationships in the social network graph using graph augmentation strategies.

Technical Framework:

  • Graph Augmentation Strategy 1: Add potential edge relationships based on node representation similarity.
  • Graph Augmentation Strategy 2: Design an edge classifier to add homophilic and heterophilic edges.
  • Combine the original graph with Strategies 1 and 2 to build a more complete social graph.
  • Apply heterogeneous GNN on the updated graph to aggregate neighbor features for node classification.

6. Social Bot Detection Based on Window Strategy.

Link: PDF

Authors: Boyu Qiao, Kun Li, Wei Zhou, Zhou Yan, and Songlin Hu.

Conference: ICME 2023 (CCF B)

Objective: Identify differences in posting behavior motivations between social bots and humans using a sliding window strategy.

Technical Framework:

  • Build initial posting embeddings using Roberta, and update with Bi-LSTM to capture past and future sequence information.
  • Use sliding windows to segment user posting sequences, and apply multi-head attention to assess representation changes across windows.
  • Integrate user metadata and window-based representation changes as node embeddings in the social graph.
  • Apply heterogeneous GNN to aggregate neighbor features for node classification.

1. Champion of the 'Explainable Social Network Community Detection' Track at the Songshan Lake AI Algorithm Challenge.

Link: PPT Code

Team Members: Boyu Qiao, Shiling Li, Kun Li, Wei Zhou.

Year: 2023.

Task Objective: Use modularity metric to partition users into different communities and explore the interpretability of the partitioning results.

Technical Framework:

  • Construct social graphs based on user forwarding relationships, initialize node embeddings in one-hot form, and update embeddings using GCN encoder.
  • Apply K-means to group nodes into clusters.
  • Design a differentiable modularity function to optimize the graph neural network as the loss function.
  • Explore the interpretability of the community divisions by mapping them to user attribute features.
